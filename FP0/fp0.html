FP0: Final Project Proposal

For my final project, I will be creating a photography portfolio of my dad's astrophotography using React.

Sketches

Prototype Version Description

My prototype version will include a basic interactive gallery of photos. Users should be able to interact with the gallery in some way to gain more information about each photo. The prototype version will also have a page that provides information about the photographer.

Final version Description

The final version will be fully themed and animated. Since the subject of the website is astrophotography, I would like to display the images in a way that makes them look like a part of a solar system. You can view photos based on the location of the subject of the photo in relation to the solar system. Once a user clicks a planet, the website should zoom in and show the actual picture as well as a brief description.

Development Process

  1. Learn how to get started with React and bundle the library with my local code.
  2. Figure out the architecture of my site. Do I want the 'About' page to be a separate page or just another section on the index page?
  3. Create all pages and files for my website. (index.html, index.js, index.css)
  4. Learn more about solar system mappings and how I can add that to the background of my website.
  5. Create interactions that will make photos generate and add photo descriptions. This will be enough for my prototype version.
  6. Code animations (if time allows). I want to create a Zooming effect on a "star" that turns into a photo and makes the background a Zoomed-in version of the larger solar system.
  7. Finalize styles.
  8. Populate data in 'About' page and style it.
  9. Deploy the final version to Github.

Unknowns

  • How to bundle React.
  • Learning React syntax.
  • How to animate with React.

Stretch Goals

Creating a website using a language that I do not know at all is already a stretch goal in my opinion. On top of that, I do want my site to be somewhat polished and usable for my dad to showcase his photos. This means that I want everything to be cleanly styled and to have animations that tie the site together. I believe this will be the hardest aspect of this project.
